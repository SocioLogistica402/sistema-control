### Sistema de Control Operativo - Logística de Sanitarios Portátiles

#### Descripción del Proyecto

Este proyecto se centra en la gestión operativa de logísticas para sanitarios portátiles. A continuación se detallan los requisitos del sistema:

1. **Gestión de Activos**: 
   - Registro de todos los sanitarios portátiles disponibles.
   - Mantenimiento y actualizaciones de cada unidad.

2. **Rutas**:
   - Planificación de rutas óptimas para la recolección y entrega.
   - Integración con mapas para visualización y seguimiento.

3. **Registros (Logs)**:
   - Registro de todas las operaciones realizadas (entregas, recolecciones, mantenimiento).
   - Generación de reportes para el seguimiento de las actividades.

4. **Gestión de Inventario**:
   - Control del inventario en tiempo real para evitar faltantes.
   - Notificaciones automáticas cuando se alcance un nivel mínimo de stock.

Este sistema busca optimizar la logística, mejorar la eficiencia del servicio y reducir costos operativos. Se implementará en fases, comenzando por la gestión de activos y rutas.
